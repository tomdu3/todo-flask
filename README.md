# Todo Flask App
[Link to the deployed site (wait a bit for the first loading)](https://todo-flask-gqvh.onrender.com/)
A simple Flask/Bootstrap Todo App

- Python 3.12.4
- Flask 3.0.3
- Werkzeug 3.0.3

- SQLAlchemy 2.0.32
